Star Wars Battle RPG

This is a project to work with Bootstrap, dynamic CSS, objects, and arrays. To play, you select a player. You will then need to battle the remaining characters. The system dynamically adjusts based upon your choices.


GitHub Repository

Doctor Who Trivia Game

This is a project to work with CSS, timing, and working with randomized arrays. Click to start and then answer the questions. You only have a limited amount of time to answer before the next question appears, so think quickly. The game play is randomized: The question order is never the same twice and the answers in the questions are also randomized.


GitHub Repository

Giphy Gallery

This is a project to work with APIs and AJAX. The site searches the Giphy repository of animated GIFs. You can filter your searches by rating and set a limit on the number of images returned. When you do a search, it adds it to your list of saved searches; just click on it again to rerun the search. You can also remove a search term from your list.


GitHub Repository

GameStart

This was a team project. The primary technologies are APIs, AJAX, and Firebase. The focus is to provide new gamers with information on how to play their games. It searches the Reddit and Twitch to find articles and people playing the game so that you can be better prepared to play the game before you start.


This project also features a chat function. Leveraging the Firebase socketing ability, it has an interactive chat section where you can talk with other gamers to get information on how to play your game.


GitHub Repository

TED2

This is another team project focusing on SQL and is deployed to Heroku and thus also uses Express. The system is seeded with information from TED's database of talks. After logging in, you can search for talks based upon a speaker or subject and it returns those talks that have been found. It allows you to view the talk in the browser and provide you with links to talks with similar subjects.


GitHub Repository

New York Times Search

This is a basic React application. It searches the New York Times API for articles based upon the subject and allows you to filter based upon the year. Click on an article to save it for later. Then go into your Saved Articles to read those articles you've saved. When done, just click on Unsave to remove it from your list.


GitHub Repository

The Fantasticks at Point Loma Playhouse (Bellomy)

The longest-running musical of all time! Harvey Jones and Tom Jones tell the story from Les Romanesques of Luisa and Matt, the children of Bellomy and Hucklebee. Their fathers are feuding and have built a wall separating their homes, but Luisa and Matt are secretly in love with each other. But surprise! The fathers aren't feuding at all! Knowing that children will always do the opposite of what you tell them to, they have decided to pretend to feud in order for their children to rebel and fall in love. But how to end the feud? Enter El Gallo, Henry, and Mortimer to stage an attack which Matt can defend against, save Luisa, and happy ending!

But is it a happy ending? As El Gallo points out, life never ends in the moonlit night.

The Fantasticks opens May 17, 2019 at Point Loma Playhouse.
